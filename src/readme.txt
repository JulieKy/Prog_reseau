
Le projet est composé de 7 fichiers.c et 5 fichiers.h

***** Dossier Client *****
 - client.c : Gére tout le côté client
 - client_tools.c : Fonctions utilisées pour le client


***** Dossier Serveur ******
 - server.c : Gère tout le côté serveur
 - server_tools.c: Fonctions directement utile pour server.c
 - user_tools.c : S'occupe de toutes les fonctions sur user (création de la structure et actions liées à cette structure)
 - cmd_tools.c: Comprend toutes les fonctions qui vont être utile lorsque le client tape une commande
